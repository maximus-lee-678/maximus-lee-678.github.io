Console-based Utility Application

Description

Windows utility to automatically download tracks from Bandcamp. Automatically performs ID3 tagging on downloaded songs.

Specify any combination of tracks, albums or artists, and send it! The program automatically retrieves the underlying MP3-128 tracks from the page source and downloads them to a single folder, fully tagged for your convienience.

READ: Bandcamp's stance on track rips. If you enjoy an artist's work, please support them by purchasing their track(s) directly. You gain access to high-quality MP3s, FLACs, and unlimited streaming!

Description

Windows utility that polls sgpokemap.com to obtain raid details at set intervals. Created since the original website does not have any built in form of notification.

Allows you to choose gyms to track, polling time interval and which raid tiers to display and notify while having a light memory footprint. Gyms can be added by name search or proximity from specified "home" coordinates.

2D Game

Description

• A multiplayer networked top-down battle royale where players battle each other in a shrinking arena. Players can enter their usernames, which are displayed above their characters and in the kill-feed.
• Each player is initially spawned at random locations throughout the map and will need to scavenge for weapons located in crates randomly spawned thorughout the map. Weapons include the SMG, Sniper, Rifle and Shotgun.
• Players seek out and eliminate one another until only one player is left standing. A gas wall that deals heavy contact damage closes in to the centre of the playable area as the match progresses, encouraging player engagement.

A labor of love for our OOP module, our team went vanilla with this project. No game development libraries were used, only basic Java Swing with a rudimentary networking system. Many a good time was had testing the game over Hamachi!

I did a lot of work on the terrain and structures, implementing their generation and collision. Many an ArrayIndexOutOfBoundsException was thrown in the process...

The terrain is implemented in quite an interesting way; it's mapped to randomised textures based on image data from a small file! Does the battleground look familiar?

Website - Full Stack

Description

An online forum website implemented as a MERN stack using tailwind.css for styling. Developed for a Secure Software Development module, this project sports a wide array of security features on both the client and server side, including Anti-CSRF, JWTs, rate limiting and more.

This website's layout was inspired in part by both Reddit (general layout) and YouTube (comments section). I had a big part in the frontend design and implementation for this project, and I have to admit that tailwind really does simplify the styling of elements. React is also an interesting beast, what with all of the state based functionality. Components are great though, wish I had some when creating this website!
